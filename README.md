# 安装

## Windows

1. 在[Releases](../../releases/latest)下载最新版
2. 解压到任意目录
3. 运行`MCST.exe`

## Linux

1. 安装[make](https://www.gnu.org/software/make), [go](https://go.dev/dl)

2. 运行以下命令

```bash
# 克隆仓库
git clone https://github.com/Arama-Vanarana/MCServerTool.git --depth 1
cd MCServerTool
sudo make install
cd ..
rm -rf MCServerTool
# 运行MCSCS
MCST
```
